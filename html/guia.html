Guía: Cómo integrar lógica JavaScript en páginas HTML

1. ¿Qué es JavaScript en HTML?

JavaScript permite hacer que las páginas HTML sean interactivas. Puedes responder a eventos, cambiar contenido, validar formularios, y mucho más.

2. Formas de integrar JavaScript en una página

A. Dentro del HTML (script embebido):

<script>
  alert('Hola desde JavaScript!');
</script>

B. Desde un archivo externo:

<script src="script.js"></script>

3. Ubicación recomendada del script

Coloca el script justo antes de cerrar la etiqueta </body> para asegurar que todos los elementos HTML ya están cargados.

4. Interacción con el DOM

El DOM (Document Object Model) es una representación estructurada del contenido HTML de una página web. JavaScript puede acceder a este modelo y manipularlo dinámicamente.

¿Para qué sirve?

  • Acceder a elementos del HTML mediante ID, clases o etiquetas.
  • Modificar contenido textual, atributos o estilos.
  • Crear y eliminar elementos dinámicamente.

¿Cómo funciona?

El navegador interpreta el HTML y construye una estructura jerárquica de nodos. JS puede usar funciones como getElementById(), querySelector() o createElement() para interactuar con estos nodos.

Ejemplo 1: Cambiar texto

Obtiene un elemento con ID "mensaje" y cambia su contenido de texto.

function cambiarTexto() {
  document.getElementById("mensaje").textContent = "Texto cambiado";
}

Ejemplo 2: Cambiar estilos

Modifica directamente el estilo de un elemento del DOM.

function cambiarColor() {
  document.getElementById("caja").style.backgroundColor = "blue";
}

Ejemplo 3: Crear un nuevo elemento

Crea un nuevo ítem de lista y lo agrega al final de una lista existente.

function agregarElemento() {
  let li = document.createElement("li");
  li.textContent = "Nuevo ítem";
  document.getElementById("lista").appendChild(li);
}

5. Escuchar eventos con JS

Un evento es una acción del usuario (clic, presionar una tecla, pasar el mouse, etc.). JavaScript puede reaccionar a estos eventos usando addEventListener.

¿Cómo funciona?

Se selecciona un elemento del DOM y se le asocia un evento. Al dispararse ese evento, se ejecuta una función.

Ejemplo 1: Evento de clic

Al hacer clic en un botón, se muestra un mensaje.

document.getElementById("boton").addEventListener("click", function() {
  alert("¡Botón presionado!");
});

Ejemplo 2: Evento de teclado

Detecta qué tecla se presiona en un campo de texto.

document.getElementById("entrada").addEventListener("keyup", function(event) {
  console.log("Tecla presionada: " + event.key);
});

Ejemplo 3: Evento al pasar el mouse

Cambia el color del fondo al pasar el mouse sobre un elemento.

document.getElementById("caja").addEventListener("mouseover", function() {
  this.style.backgroundColor = "green";
});

6. Buenas prácticas

  • Separar HTML, CSS y JS en archivos distintos.
  • Usar nombres claros para funciones e IDs.
  • Evitar código duplicado.
  • Probar en varios navegadores.

7. Ejemplo completo


<!DOCTYPE html>
<html>
<head>
  <title>Mi página JS</title>
</head>
<body>
  <h1 id="saludo">Hola Mundo</h1>
  <button onclick="saludar()">Haz clic</button>
  <script src="script.js"></script>
</body>
</html>
  
// script.js
function saludar() {
  document.getElementById("saludo").textContent = "Hola desde JS externo!";
}
